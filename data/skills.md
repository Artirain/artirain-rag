# Навыки и стек

## Языки и основное
- Python (основной): AsyncIO, dataclasses, subprocess, pytest.
- SQL, работа с SQLite и Postgres.

## LLM / AI
- Провайдеры: OpenAI API, Anthropic Claude, Codex CLI (gpt-5.5).
- Паттерны: классификация, извлечение структуры из текста, генерация, RAG
  (retrieval + подмешивание контекста), агентная маршрутизация, фоллбэк между движками.
- Эмбеддинги и векторный поиск: fastembed, Qdrant.
- Промпт-инжиниринг, eval-метрики (hit@k, latency, токены).

## Интеграции и автоматизация
- REST API (job boards, e-commerce платформы, банковские/трекинговые сервисы), OAuth.
- Playwright (browser-автоматизация), парсинг.
- Telegram-боты (Telethon, Bot API), нотификации.
- Планировщики: Windows Task Scheduler, cron.

## Инфраструктура и DevOps
- Docker, docker-compose.
- Kubernetes (Helm).
- CI/CD: GitLab CI/CD, GitHub Actions.
