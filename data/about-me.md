# О кандидате

Python-разработчик с уклоном в AI/LLM-инженерию и автоматизацию. Более года поддерживаю
production-среду e-commerce-проекта (розничная сеть, несколько точек): ~20 микросервисов
в проде. Строю end-to-end сервисы: интеграции с внешними API, LLM-пайплайны (классификация,
извлечение структуры, генерация текста), автоматизация бизнес-процессов, нотификации в Telegram.

## Направления
- LLM-приложения: классификация, извлечение структуры из текста, генерация,
  ретрив-подмешивание контекста, агентные сценарии с маршрутизацией.
- Бэкенд на Python: интеграции REST API, асинхронность (AsyncIO), очереди задач,
  состояние в SQLite/Postgres.
- Автоматизация: Playwright (browser-flow), парсинг, планировщики (Task Scheduler,
  cron), Telegram-боты.
- Деплой: Docker, docker-compose, Kubernetes (Helm), GitLab CI/CD, GitHub Actions.

## Подход к работе
- TDD: сначала тесты (pytest), затем код.
- Простые решения вместо over-engineering.
- Секреты только в .env, никогда в git.
